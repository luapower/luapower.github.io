Math | requires: for 2D rectangles defined as (x, y, w, h).

---------------------------------------------------------------------------------------------- | requires: -------------------------------------------- representation | requires: forms box.corners(x, | requires: y, w, h) -> x1, y1, x2, y2 left,top and right,bottom corners box.rect(x1, | requires: y1, x2, y2) -> x, y, w, h box given left,top and right,bottom corners layouting box.align(w, | requires: h, halign, valign, bx, by, bw, bh) -> x, y, w, h align a box in another box box.vsplit(i, | requires: sh, x, y, w, h) -> x, y, w, h slice a box horizontally at a certain height and return the i'th box. if sh is negative, slicing is done from the bottom side. box.hsplit(i, | requires: sw, x, y, w, h) -> x, y, w, h slice a box vertically at a certain width and return the i'th box. if sw is negative, slicing is done from the right side. box.nsplit(i, | requires: n, direction, x, y, w, h) -> x, y, w, h slice a box in n equal slices, vertically or horizontally, and return the i'th box. direction = 'v' or 'h' box.translate(x0, | requires: y0, x, y, w, h) -> x, y, w, h move a box box.offset(d, | requires: x, y, w, h) -> x, y, w, h offset a box by d, outward if d is positive box.fit(w, | requires: h, bw, bh) -> w, h fit a box into another box preserving aspect ratio. use align() to position the box hit | requires: testing box.hit(x0, | requires: y0, x, y, w, h) -> true | false check if a point (x0, y0) is inside rect (x, y, w, h) box.hit_margins(x0, | requires: y0, d, x, y, w, h) -> hit, left, top, right, bottom hit test for margins and corners edge | requires: snapping box.snap_margins(d, | requires: x, y, w, h, rectangles) -> x, y, w, h snap the sides of a rectangle against an iterator of rectangles. rectangles = function() -> x, y, w, h | nil box.snap_pos(d, | requires: x, y, w, h, rectangles) -> x, y, w, h snap the position of a rectangle against an iterator of rectangles. box.snapped_margins(d, | requires: x1, y1, w1, h1, x2, y2, w2, h2) -> snapped, left, top, right, bottom check if two boxes are snapped and on which margins. ---------------------------------------------------------------------------------------------- | requires:
