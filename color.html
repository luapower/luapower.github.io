v1.0 | requires: | Lua 5.1, Lua 5.2, LuaJIT 2

| requires: local color = require'color'

Color | requires: computation in HSL space. Shamelessly ripped off and modified from [Sputnik's colors lib][colors lib], by Yuri Takhteyev.

| requires: * r, g, b, s, L are in 0..1 range. | requires: * h is in 0..360 range.

| requires: Direct conversions

---------------------------------------------------- | requires: ------------------------------------------------ color.hsl_to_rgb(h, | requires: s, L) -> r, g, b HSL -> RGB; h is modulo 360; s, L are clamped to range color.rgb_to_hsl(r, | requires: g, b) -> h, s, L RGB -> HSL; r, g, b are clamped to range color.rgb_to_string(r, | requires: g, b) -> s generate '#rrggbb' hex color color.string_to_rgb(s) | requires: -> r, g, b | nil parse a '#rrggbb' hex color color.rgba_to_string(r, | requires: g, b, a) -> s generate a '#rrggbbaa' hex color color.string_to_rgba(s) | requires: -> r, g, b, a | nil parse a '#rrggbbaa' hex color (the 'aa' part is optional) ---------------------------------------------------- | requires: ------------------------------------------------

| requires: Color objects

---------------------------------------------------- | requires: ------------------------------------------------ color('#rrggbb') | requires: -> col create a new HSL color object from a RGB string color(h, | requires: s, L) -> col create a new HSL color object from HSL values col.h, | requires: col.s, col.L color fields (for reading and writing) col:hsl() | requires: -> h, s, L
col() -> h, s, L color fields unpacked col:rgb() | requires: -> r, g, b convert to RGB col:tostring() | requires: -> '#rrggbb' convert to RGB string col:hue_offset(hue_delta) | requires: -> color create a new color with a different hue col:complementary() | requires: -> color create a complementary color col:neighbors(angle) | requires: -> color1, color2 create two neighboring colors (by hue), offset by "angle" col:triadic() | requires: -> color1, color2 create two new colors to make a triadic color scheme col:split_complementary(angle) | requires: -> color1, color2 create two new colors, offset by angle from a color's complementary col:desaturate_to(saturation) | requires: -> color create a new color with saturation set to a new value col:desaturate_by(r) | requires: -> color create a new color with saturation set to a old saturation times r col:lighten_to(lightness) | requires: -> color create a new color with lightness set to a new value col:lighten_by(r) | requires: -> color create a new color with lightness set to its lightness times r col:variations(f, | requires: n) -> {color1, ...} create n variations of a color using supplied function and return them as a table col:tints(n) | requires: -> {color1, ...} create n tints of a color and return them as a table col:shades(n) | requires: -> {color1, ...} create n shades of a color and return them as a table col:tint(r) | requires: -> color create a color tint col:shade(r) | requires: -> color create a color shade ---------------------------------------------------- | requires: ------------------------------------------------
